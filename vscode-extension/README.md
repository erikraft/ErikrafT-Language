# ErikrafT Language Support for VS Code

Professional support for the ErikrafT language (.erik).

## Features

- **Syntax Highlighting**: Complete support for ErikrafT keywords, types, and AI primitives.
- **IntelliSense**: Smart completion for language constructs and `ai.*` methods.
- **Hover Information**: Documentation for built-in functions.
- **Diagnostics**: Real-time syntax checking and bracket matching.
- **Formatting**: Automatic code formatting on save or command.
- **Snippets**: Productivity shortcuts for common ErikrafT patterns.
